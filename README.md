# Kit RPG Web

Una aplicación web simple desarrollada con **HTML, CSS y JavaScript** que simula herramientas básicas de un juego de rol (RPG).

## Estructura del Proyecto

```text
KitRPGWeb/
│
├── index.html
├── style.css
└── script.js
```


### 2. Acceder al proyecto

```bash
cd kit-rpg-web
```

### 3. Ejecutar la aplicación

Abrir el archivo:

```text
index.html
```
